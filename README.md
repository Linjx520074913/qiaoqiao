# 智能自动记账 App (Bill)

这是一个基于 SwiftUI 开发的 iOS 自动记账应用程序，支持智能识别和自动分类记账功能。

## 功能特性

### 1. 记账主页 (HomeView)
- 用户问候和自动记账状态显示
- 多账户支持（全部账户、现金、银行卡等）
- 账户资产卡片展示
  - 显示总资产余额
  - 显示本月收入和支出
  - 支持隐藏/显示余额
  - 精美的渐变背景设计
- 快捷操作按钮：记支出、记收入、转账
- 最近记录列表
  - 显示分类图标和颜色
  - 支持8大分类（餐饮、购物、交通、娱乐、住房、医疗、教育、其他）
  - 显示交易金额（收入为绿色，支出为红色）
  - 交易时间和描述

### 2. 账单统计页面 (StatisticsView)
- 时间段切换（周/月/年）
- 动态圆环图表
  - 显示不同类别的消费占比
  - 支持8大分类，每个分类有独特的颜色和图标
  - 彩色编码的图例
- 支出明细列表展示

### 3. 我的账本页面 (ProfileView)
- 用户头像和昵称编辑
- 账本设置卡片
  - 账本类型选择
  - 记账提醒设置
  - iCloud 同步功能
- 财务概览
  - 总收入和总支出统计
  - 本月预算使用进度
  - 预算剩余提示
- 智能自动记账开关
  - 渐变色卡片设计
  - 一键启用/关闭自动记账
- 功能说明

### 4. 底部导航栏 (MainTabView)
- 5 个标签：记账、统计、扫描、账本、我的
- 中间浮动扫描按钮（用于扫描小票自动记账）
- 自定义图标和颜色

1. 在 Xcode 中打开 `bill.xcodeproj`
2. 选择目标设备或模拟器
3. 点击运行按钮 (⌘+R)

## 主要组件

### 数据模型
- `Transaction`: 交易记录
- `Account`: 账户/账本信息
- `UserProfile`: 用户信息
- `StatisticsData`: 统计数据
- `TransactionCategory`: 记账分类枚举（餐饮、购物、交通、娱乐、住房、医疗、教育、其他）
- `TransactionType`: 交易类型（收入/支出）

### 自定义组件
- `AccountCardView`: 账户卡片展示组件
- `QuickActionButton`: 快捷操作按钮
- `TransactionRow`: 交易记录行视图（带分类标签和彩色图标）
- `DonutChart`: 圆环统计图表
- `ProfileInfoRow`: 设置信息行
- `AutoRecordCard`: 自动记账功能卡片
- `CustomTabBar`: 自定义底部导航栏
