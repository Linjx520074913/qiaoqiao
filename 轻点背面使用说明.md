# 轻点背面使用说明

## 🎯 问题：敲击后背进入App，什么都没有发生

### 原因分析

当你使用iPhone的"轻点背面"功能时：
1. ⚡️ 轻点背面触发快捷指令
2. 📸 快捷指令开始截屏
3. 🚀 App被打开
4. ❌ **但截图还没生成！**

这就是为什么App打开后看不到图片。

## ✅ 解决方案（已实现）

现在App会**智能等待**截图：
- App打开后**持续监听剪贴板3秒**
- 每0.5秒检查一次
- 一旦检测到图片，立即打开解析页面
- 完美解决时序问题！

## 📱 推荐的快捷指令配置

### 方案1：使用剪贴板（最稳定）⭐️⭐️⭐️⭐️⭐️

```
步骤1: 截屏
步骤2: 从 截屏 获取图像
步骤3: 拷贝到剪贴板
步骤4: 打开 bill
```

**为什么推荐？**
- ✅ App会等待3秒检测剪贴板
- ✅ 即使截图慢一点也能检测到
- ✅ 最稳定可靠

### 方案2：添加等待时间

如果你坚持不用剪贴板，可以这样：

```
步骤1: 截屏
步骤2: 等待 1秒
步骤3: 从 截屏 获取图像
步骤4: 编码（Base64）
步骤5: 打开URL: bill://[Base64编码的结果]
```

**添加"等待"步骤**：
- 搜索"等待"
- 设置等待时间：1秒
- 确保截图完成

## 🎮 配置轻点背面

### 1. 打开设置
设置 → 辅助功能 → 触控 → 轻点背面

### 2. 选择轻点次数
- **轻点两下**：适合日常使用
- **轻点三下**：避免误触

### 3. 选择你的快捷指令
- 滑到最下方"快捷指令"区域
- 选择你创建的识别账单快捷指令

## ⚡️ 完整工作流程

```
轻点iPhone背面
    ↓
触发快捷指令
    ↓
截屏 + 获取图像 + 拷贝到剪贴板
    ↓
打开bill App
    ↓
App开始监听剪贴板（持续3秒）
    ↓
检测到图片！
    ↓
自动打开解析页面
    ↓
显示图片
    ↓
点击"解析账单"
    ↓
完成！
```

## 🔍 如何知道是否检测到图片？

### 成功的表现：
1. App打开
2. 等待0.5-2秒
3. **自动弹出图像解析页面**
4. 看到你的截图

### 失败的表现：
1. App打开
2. 停留在首页
3. 3秒后没有反应

### 失败了怎么办？

**立即解决方案**：
1. 点击底部中间的蓝色扫描按钮
2. 从相册选择刚才的截图
3. 点击"解析账单"

**长期解决方案**：
- 确保快捷指令有"拷贝到剪贴板"步骤
- 或者添加1秒等待时间

## 🐛 调试技巧

### 问题1：轻点背面没反应
**检查**：
- 设置 → 辅助功能 → 触控 → 轻点背面
- 确认已绑定快捷指令
- 尝试用力一点轻点

### 问题2：App打开了但没图片
**原因**：剪贴板没有图片
**解决**：
- 检查快捷指令是否有"拷贝到剪贴板"
- 或手动点击扫描按钮选择图片

### 问题3：有时候可以有时候不行
**原因**：时序问题
**解决**：
- 在快捷指令中添加"等待 1秒"
- 或使用剪贴板方案
