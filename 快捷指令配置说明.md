# 快捷指令配置说明

## 功能概述

本应用支持通过 iOS 快捷指令自动识别账单图片，实现：
- 📸 截屏获取小票/账单图像
- 🔍 自动OCR文字识别
- 🤖 智能解析账单信息（商户、金额、分类）
- 📊 在调试区域查看识别结果
- 💾 一键保存到账单列表

## 快捷指令配置步骤

### 方法一：使用URL Scheme（推荐）

1. **打开快捷指令App**
2. **创建新快捷指令**
3. **添加以下操作**：

   ```
   操作1: 获取最新的屏幕截图
   操作2: 对图像进行Base64编码
   操作3: 打开URL
          - URL格式：bill://parse-image?data=[Base64编码的结果]
   ```

### 方法二：使用剪贴板传递图片（最简单）⭐️

1. **打开快捷指令App**
2. **创建新快捷指令**
3. **添加以下操作**：

   ```
   操作1: 截屏
   操作2: 从 截屏 获取图像
   操作3: 拷贝到剪贴板
   操作4: 打开 bill
   ```

4. **App会自动**：
   - 检测剪贴板中的图片
   - 自动加载到解析页面
   - 点击"解析账单"即可识别

### 方法三：保存到相册后手动选择

1. **打开快捷指令App**
2. **创建新快捷指令**
3. **添加以下操作**：

   ```
   操作1: 获取最新的屏幕截图
   操作2: 存储到相册
   操作3: 打开App "bill"
   ```

4. **在App中**：
   - 点击底部中间的扫描按钮
   - 选择刚才保存的图片
   - 点击"解析账单"按钮

## 详细配置示例（方法一）

### 步骤1：获取屏幕截图
- 添加操作："获取屏幕截图"
- 设置为"最新的屏幕截图"

### 步骤2：编码图像
- 添加操作："编码"
- 编码方式：Base64
- 输入：屏幕截图

### 步骤3：打开URL
- 添加操作："打开URL"
- URL填写：`bill://parse-image?data=`
- 在data=后面选择"Base64编码的结果"

## 使用流程

1. **截取小票图片**
   - 对准小票/账单进行截屏（Home + 电源键 或 侧边按钮）

2. **运行快捷指令**
   - 运行配置好的快捷指令

3. **查看识别结果**
   - App自动打开图像解析页面
   - OCR识别区域显示提取的所有文字
   - 账单信息区域显示解析的结构化数据

4. **调试信息说明**

   **OCR识别结果区域**：
   - 显示图像中识别出的所有文字
   - 使用等宽字体，保持原始格式
   - 可滚动查看完整内容

   **解析的账单信息区域**：
   - 商户名称：自动识别的商家
   - 金额：提取的消费金额
   - 类型：支出/收入
   - 分类：智能分类（餐饮、购物、交通等8大类）
   - 描述：附加说明
   - 时间：识别时间戳

5. **保存账单**
   - 确认信息无误后，点击"保存账单"按钮

## 调试技巧

1. **识别不准确**：
   - 确保小票/账单图像清晰
   - 文字不要倾斜太多
   - 光线充足，对比度高

2. **分类错误**：
   - 在调试区域查看OCR识别结果
   - 检查是否包含分类关键词
   - 可手动调整分类（保存前）

3. **金额识别失败**：
   - 确保金额数字清晰可见
   - 检查是否使用支持的格式
   - 在调试区域查看原始文字

## URL Scheme说明

**格式**：`bill://parse-image?data=<base64_image_data>`

**参数**：
- `data`：Base64编码的图片数据（可选）
- 如果不提供data参数，会直接打开图像解析页面，需手动选择图片

**示例**：
```
bill://parse-image
bill://parse-image?data=iVBORw0KGgoAAAANS...
```

## 常见问题

**Q: 快捷指令运行后没有反应？**
A: 检查URL Scheme是否正确配置，确保使用 `bill://` 而不是 `bills://`

**Q: OCR识别不出中文？**
A: 应用已配置支持简体中文和英文，确保iOS系统语言设置正确

**Q: 能识别手写小票吗？**
A: 系统OCR对打印文字识别效果最好，手写字体识别率较低

**Q: 能批量识别多张小票吗？**
A: 当前版本仅支持单张识别，批量功能将在后续版本添加

## 技术实现

- **识别语言**：简体中文 + 英文
- **识别精度**：accurate 模式
- **URL Scheme**：bill://
- **图片传输**：Base64编码
