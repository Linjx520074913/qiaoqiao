# 🔍 调试模式使用说明

## 问题：不知道发生了什么，也不知道哪里有错误

### ✅ 解决方案：内置调试面板

现在App有一个**隐藏的调试面板**，可以看到所有运行信息！

## 📱 如何打开调试面板

### 方法1：自动弹出（推荐）

如果快捷指令运行后3秒内没有检测到图片，调试面板会**自动弹出**，显示详细信息。

### 方法2：手动打开

在App主界面，**从上往下滑动**（任意位置），调试面板会弹出。

## 🔧 根据调试信息诊断问题

### 问题1：看到"监听3秒未发现图片"

**原因**：快捷指令没有把图片复制到剪贴板

**解决方案**：
```
检查你的快捷指令，必须有这些步骤：
1. 截屏
2. 从截屏获取图像
3. 拷贝到剪贴板        ← 必须有这步！
4. 打开 bill
```

### 问题2：看到"剪贴板中发现图片内容"但之后是"获取失败"

**原因**：图片格式问题

**解决方案**：
- 重新配置快捷指令
- 确保"获取图像"步骤正确
- 尝试重新截图

### 问题3：根本没有看到任何消息

**原因**：App没有启动或崩溃了

**解决方案**：
- 重新安装App
- 检查iOS版本（需要iOS 18.5+）
- 在Xcode中运行查看错误

## 📋 完整的诊断流程

### 第1步：运行快捷指令
- 轻点iPhone背面
- 或手动运行快捷指令

### 第2步：等待3秒
- App会自动监听剪贴板
- 不要做任何操作

### 第3步：查看调试面板
- 如果没有自动弹出，向下滑动打开
- 阅读最新的消息

### 第4步：根据消息修复问题

#### 看到"✅ 成功获取图片"
→ 一切正常！等待解析页面打开

#### 看到"⚠️ 监听3秒未发现图片"
→ 快捷指令缺少"拷贝到剪贴板"步骤

#### 看到"❌ 获取失败"
→ 图片格式问题，重新配置快捷指令

#### 什么都没看到
→ App可能崩溃，检查Xcode日志

## 💡 常见问题排查

### Q1: 调试面板显示"未发现图片"怎么办？

**检查清单**：
- [ ] 快捷指令有"拷贝到剪贴板"步骤
- [ ] "拷贝到剪贴板"的输入是"图像"
- [ ] 快捷指令成功运行（没有错误）
- [ ] 截图成功（可以在相册看到）

**修复步骤**：
```
打开快捷指令App
→ 编辑你的快捷指令
→ 在"打开 bill"之前添加
→ "拷贝到剪贴板"操作
→ 输入选择"图像"
→ 保存
```

### Q2: 调试面板自动弹出，说明什么？

说明：**3秒内没有检测到剪贴板中的图片**

这是正常的提示，表示需要你：
1. 检查快捷指令配置
2. 添加"拷贝到剪贴板"步骤
3. 或手动点击扫描按钮选择图片

### Q3: 看到"成功获取图片"但解析页面没打开？

**可能原因**：
1. UI加载延迟
2. 内存不足
3. iOS版本问题

**解决方案**：
- 等待1-2秒
- 重启App
- 手动点击扫描按钮

## 🚀 推荐的测试流程

### 第一次使用建议：

1. **准备工作**
   - 打开一张小票或账单
   - 确保光线充足

2. **截图测试**
   - 先手动截图
   - 看调试面板显示什么

3. **配置快捷指令**
   ```
   截屏
   → 从截屏获取图像
   → 拷贝到剪贴板  ← 关键
   → 打开 bill
   ```

4. **运行测试**
   - 轻点背面运行快捷指令
   - 观察调试面板消息
   - 根据提示调整

5. **成功标志**
   - 看到"✅ 成功获取图片"
   - 解析页面自动打开
   - 显示截图
